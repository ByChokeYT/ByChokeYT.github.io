About Me Projects

My Story

Over the past three years as a computer systems engineer, I have designed and developed scalable web applications using modern technologies such as HTML, CSS, JavaScript, Next.js, React, and Node.js. My work includes managing production and development servers as well as providing data analysis and statistics in conjunction with data teams. I am also responsible for maintaining web applications and databases, and I use analysis tools to identify patterns and trends in the data. I have developed prediction models and manipulated data using programming languages such as R, Python, and SQL. I have also integrated data into web applications in collaboration with web development teams. One of my most notable projects was designing and implementing an inventory management application using the .NET Framework and C#. I used ASP.NET to improve system efficiency through real-time inventory updates and low inventory alerts, and I also integrated the application with a SQL database and third-party API for processing payments. I performed performance and stability tests to encdsure the scalability of the system.
